# soul-os-docs

Documentation site for Soul OS, built with Vite + React on the frontend and a small Express server for production hosting.

## Prerequisites

- Node.js 20+
- pnpm 10.4.1 (via Corepack)

## Getting Started

```bash
corepack enable
corepack prepare pnpm@10.4.1 --activate
pnpm install
```

## Development

```bash
pnpm dev
```

## Build and Run

```bash
pnpm build
pnpm start
```

## Quality Checks

```bash
pnpm check
pnpm format
```
